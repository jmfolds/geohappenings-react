# Geohappenings - React edition

## Requirements

  * [Node.js](http://nodejs.org): Use the installer provided on the NodeJS website.

## Quickstart

```bash
git clone git@github.com:jmfolds/geohappenings-react.git

cd geohappenings-react

npm install

npm run serve --> builds and starts dev server at localhost:9000
npm run start --> watch files for changes and re-compile

```
